## UiKit

UiKit проекта Realt One

## Кодогенерация при добавлении иконок или цветов
Можно переключиться в директорию пакета ui_kit и выполнить flutter pub get или команда из любой директории
```
fluttergen -c YOUR_PATH_TO_PROJECT/realt-one/packages/ui_kit/pubspec.yaml

Пример:
fluttergen -c /Users/brntsv/projects/nan/realt-one/packages/ui_kit/pubspec.yaml
```
